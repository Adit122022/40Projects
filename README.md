## **Setup Instructions**

Each task is in its own folder (e.g., `task1`, `task2`). To run a specific project:

1. Navigate to the project folder:
   ```bash
   cd task1
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
